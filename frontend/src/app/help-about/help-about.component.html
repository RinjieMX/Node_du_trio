How Anki Works

Introduction

Anki is a spaced repetition application that uses flashcards to facilitate learning and memorization.

Basic Principle

Anki employs the spaced repetition algorithm to schedule facts reviews at optimal times, maximizing long-term retention.

Creating Packages

Users can create packages for certain subjects and precise the level of difficulty, what audience was it made for. These packages are then reviewed based on the user's mastery level.

Creating Facts

Users can create facts containing questions and answers. These cards are then reviewed based on the user's mastery level.

  • Questions on the front, answers on the back.
  • Use of media (images, audio, video) to enhance cards.

Daily Review

Anki offers a daily review of facts, automatically adjusting the repetition interval based on the user's performance.
