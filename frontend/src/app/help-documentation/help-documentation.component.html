Learn and Explore

If you want to learn more by yourself, here are some websites that could be usefull for each subject.

And remember, practice makes perfect!

Maths

  • Khan Academy - Provides a wide range of free online courses and lessons in various math topics.
  • Mathway - A problem solver that can answer math problems in various areas, offering step-by-step solutions.
  • Wolfram Alpha - A computational search engine that provides answers and solutions to a wide range of mathematical problems.
  • Math Is Fun - An interactive site offering clear explanations, games, and puzzles to make learning math enjoyable.

English

  • Duolingo - Interactive language learning platform.
  • BBC Learning English - English language lessons from the BBC.

Physics

  • Khan Academy - Physics - Comprehensive physics lessons and tutorials for various topics.
  • PhET Interactive Simulations - Interactive math and science simulations for physics concepts.
  • Physics Classroom - Online tutorials and resources covering a wide range of physics topics.

Philosophy

  • Stanford Encyclopedia of Philosophy - A comprehensive and authoritative reference for philosophy.
  • Internet Encyclopedia of Philosophy (IEP) - An online encyclopedia of philosophy with articles written by experts.
  • PhilPapers - A comprehensive index and bibliography of philosophy articles.

History

  • History Files: France - Comprehensive historical information about France.
  • History: U.S. Presidents - Explore the history of the United States and its presidents.
  • History Extra: Medieval England - Articles and resources on medieval England.
  • Life in Italy: History - Information about the history and culture of Italy.
  • History Files: Japan - Comprehensive historical information about Japan.
  • Korea.net: History of Korea - Explore the history of Korea.
  • Culture.pl: A Foreigner's Guide to Polish History - Learn about the history of Poland.
  • Discover Tunisia: History - Information about the history of Tunisia.
  • Tour Egypt: Ancient Egypt - Explore the history of ancient Egypt.
  • History Files: Brazil - Comprehensive historical information about Brazil.
  • China Highlights: History - Learn about the history of China.

Geography

  • Africa - Explore news, business, and lifestyle in Africa.
  • Asia - Travel guides for Asian countries.
  • Europe - Official website of the European Union.
  • North America - Explore U.S. national parks.
  • South America - Travel and tourism information for South America.
  • Australia - Official tourism website for Australia.
  • Antarctica - Information about Antarctica from Australia.
  • World Atlas - Seas - Explore seas around the world.
  • NOAA Ocean Facts - Learn interesting facts about the world's oceans.
  • Marine Insight - List of Seas - A comprehensive list of seas around the globe.
  • National Geographic - Ocean - Discover information about the ocean from National Geographic.

Informatics

  • Codecademy Python Course - Interactive Python course.
  • Python Official Tutorial - Python documentation with a tutorial.
  • C# Tutorial - C# tutorial from Microsoft.
  • C++ Tutorial - C++ tutorial from cplusplus.com.
  • W3Schools HTML Tutorial - HTML tutorial from W3Schools.
  • W3Schools CSS Tutorial - CSS tutorial from W3Schools.
  • Java Documentation - Official documentation for Java.
  • MDN JavaScript Guide - JavaScript guide from Mozilla Developer Network (MDN).
  • TypeScript Documentation - Official documentation for TypeScript.
  • Scratch - Create stories, games, and animations with Scratch.
